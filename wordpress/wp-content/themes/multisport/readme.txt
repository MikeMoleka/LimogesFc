== Installation ==

Manual installation:

1. Upload the theme folder to the /wp-content/themes/ directory

Installation using 'Add New Theme'

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for theme name
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options


The MultiSport Theme is very suitable if you want to create a website about equipments review. Actually the screenshot.png image is an example of such type of website.

Here is a guide how to create website content like the content from theme screenshot:

Create About Me Section

1. Navigate to Admin Panel -> Left Menu -> Pages -> Add New

2. Create a new website page with information about you. Then save the page and copy page URL, i.e. http://yoursite.com/about-me.

1. Navigate to Admin Panel -> Left Menu -> Media -> Add New

2. Upload an about me image

3. Once the images is uploaded, click on it and copy its URL, i.e. http://yoursite.com/wp-content/uploads/2017/04/about.jpg

4. Navigate to Admin Panel -> Left Menu -> Appearance -> Widgets

5. Insert 'Text Widget' into 'Sidebar Widget Area'

6. Fill 'About me' for the Title field.

7. For 'Content' insert the following code:

<p><img src="http://yoursite.com/wp-content/uploads/2017/04/about.jpg" style="width:200px;float:left;margin:0 10px 10px 0;" /> Some about me text to introduce yourself.
<a href="http://yoursite.com/about-me" class="btn">Read more</a>
</p>

8. Save Changes

Create a New Post

1. Navigate to Admin Panel -> Left Menu -> Posts -> Add New

2. Create a new website page with information about the purpose of your website i.e. equipment reviews, training tips, etc.

3. You can insert an image (like the image from screenshot.png) by clicking on 'Featured Image' -> 'Set featured image' link, select or upload an new image.

4. You can enrich the page content by setting parts of it to Bold, Italic, Blockquote, Bulleted List, Numbered List, etc.

5. Save changes
